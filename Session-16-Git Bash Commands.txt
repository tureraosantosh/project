-------------------------------------------------------------------------
git init : To initialize our folder as git operations folder

git add <filename> : To add one file to git staging area

Note: The files which are added to staging area are eligible for commit.

git status: To see files added and not added to staging area

git commit -m <msg> : To commit files from staging area to git local repo

git push : To move changes from local repo to central repository.

git pull : To take changes from git central repo to local repo
